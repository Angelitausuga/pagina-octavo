BIOLOGÍA

El ciclo menstrual o ciclo ovárico consiste en una serie de cambios regulares que de forma natural ocurren en el sistema reproductor femenino los cuales hacen posible el embarazo o la menstruación, en caso de que el primero no tenga lugar, durante este ciclo se desarrollan los ovocitos.

Fases del Ciclo Menstrual

El ciclo menstrual se divide en cuatro fases:

1. Fase Menstrual: Durante esta fase, el cuerpo elimina el endometrio que se ha preparado para un posible embarazo. Esta fase suele durar alrededor de 5-7 días.

2. Fase Folículo: Durante esta fase, el cuerpo produce FSH para estimular el crecimiento de los folículos en los ovarios. Esta fase suele durar alrededor de 10-14 días.

3. Fase Ovulatoria: Durante esta fase, el cuerpo produce LH para desencadenar la ovulación. Esta fase suele durar alrededor de 24 horas.

4. Fase Lútea: Durante esta fase, el cuerpo produce progesterona para mantener el endometrio preparado para un posible embarazo. Esta fase suele durar a alrededor de 10-14 días.

Hormonas y su Función

Las hormonas involucradas en el ciclo menstrual tienen funciones específicas:

- Estrógeno: Regula el crecimiento del endometrio y prepara el cuerpo para un posible embarazo.

- Progesterona: Mantiene el endometrio preparado para un posible embarazo y apoya el embarazo.

- FSH: Estimula el crecimiento de los folículos en los ovarios.

- LH: Desencadena la ovulación y estimula la producción de progesterona.

- Prolactina: Regula la producción de leche materna durante el embarazo y la lactancia.

- Inhibina: Regula la producción de FSH y LH.

- Relaxina: Prepara el útero para el embarazo y el parto.

- Oxitocina: Estimula las contracciones uterinas durante el parto.
